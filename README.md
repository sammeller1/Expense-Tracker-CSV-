#  Expense Tracker (CLI in Python)

Track and log your expenses using a simple command-line interface. All data is saved in a structured CSV file with timestamps and categories.

---

##  Features

- Add new expenses with category and note
- Timestamp for each entry
- Stored in `expenses.csv`
- View total and detailed logs
- No external libraries required

---

## How to Run

```bash
python expense_tracker.py
